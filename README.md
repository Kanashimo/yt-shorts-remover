# YouTube Shorts Remover
Finally, an extension both for Chrome and Firefox, that keeps your brain from turning into a soft‑boiled egg every afternoon. Here is it, YouTube Shorts Remover!
## Preview
<img src="/.github/preview.png" alt="Preview" width="400">

## Installation

#### Chrome
1. Download .zip archive with source code from the [latest release](https://github.com/kanashimo/yt-shorts-remover/releases/latest)
2. Unpack it
3. Go to `chrome://extensions`
4. Enable **developer mode**
5. Click **Load unpacked** and select the unpacked folder with source code
#### Firefox
1. Download .zip archive with source code from the [latest release](https://github.com/kanashimo/yt-shorts-remover/releases/latest)
2. Unpack it
3. Go to `about:debugging#/runtime/this-firefox`
4. Click **Load temporary extension** and select the unpacked folder with source code

Have fun! Maybe in the future, I will add it to Chrome Web Store and Firefox Add-ons.
